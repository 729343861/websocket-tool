# websocket-tool

#### 介绍
一款极简好用的ws调试工具

#### 软件架构
软件架构说明
node  + uniapp

#### 功能介绍

1.  可进行ws、wss连接调试 
2.  支持多个ws客户端连接池，收发消息分析
3.  支持自定义发送ping消息
4.  本地缓存记录历史调试ws_url 及使用次数
5.  对消息进行json解码显示
6.  暂停消息显示，可提供分析问题
7.  只显示最新推送消息，可提供分析问题
8.  缓存发送历史
9.  重发指定历史消息
10.  清空消息
